# Pattern — Evidence Package

**Weight**: Recommended

## Context

Investigation needs to produce complete, reproducible evidence.

## Problem

How to organize evidence so it is complete, reproducible, and auditable?

## Forces

- Must be complete
- Must be reproducible
- Must be auditable
- Must be organized

## Solution

1. Define Evidence Structure
2. Collect Evidence per Category
3. Validate Completeness
4. Consolidate Findings
5. Prepare Candidate Decisions
6. Create Final Report

## Expected Outputs

- Evidence Package (complete)
- Findings summary
- Candidate decisions
- Final report

## Known Variations

- Technology Assessment Package
- Architecture Assessment Package
- Security Assessment Package
- Performance Assessment Package

## Anti-patterns

- Incomplete evidence
- Not organizing by category
- Missing consolidated report
- Not validating completeness
