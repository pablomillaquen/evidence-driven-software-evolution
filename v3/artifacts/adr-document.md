# Artifact — ADR Document

**Weight**: Informative

## Purpose

Documented representation of an Architectural Decision.

## Represents

System-wide design choices with full context and alternatives.

## Format

```markdown
# ADR-XXX: [Title]

**Status**: Accepted
**Date**: [Date]
**Deciders**: [Who was involved]

## Context
[What is the issue that motivates this decision?]

## Decision
[What is the change being proposed or decided?]

## Consequences
### Positive
- [Benefits]
### Negative
- [Trade-offs]
### Neutral
- [Things to be aware of]

## Alternatives Considered
### [Option A]
[Description]
[Why not chosen]
```

## Naming

`docs/adr/NNN-short-title.md`
