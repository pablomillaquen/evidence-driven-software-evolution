# Principle — Freeze Before Evolution

**Weight**: Recommended

## Description

Capture system state before introducing changes.

## When to Apply

- Before implementation begins
- After Evidence Package complete

## Rules

- All evidence must be complete before freeze
- All candidate decisions must be accepted or deferred
- Snapshot must be created
- Configuration must be captured
