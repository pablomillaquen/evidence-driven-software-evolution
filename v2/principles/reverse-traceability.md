# Principle — Reverse Traceability

**Weight**: Recommended

## Description

Each finding must be traceable back to the evidence that supports it and the Research Question that originated it.

## When to Apply

- When documenting findings
- When making decisions
- When auditing decisions

## Rules

- Finding → Evidence → Research Question
- Every decision must be auditable back to evidence
- Enable "why" questions to be answered
