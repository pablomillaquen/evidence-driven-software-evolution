# Artifact — Release Report

**Weight**: Informative

## Purpose

Documents version release.

## Represents

Summary of work completed, decisions made, and knowledge produced.

## Sections

1. Executive Summary
2. Objectives
3. Scope (Included / Not Included)
4. Work Completed (by phase)
5. Metrics (regression + performance)
6. Findings (closed + deferred)
7. Architectural Decisions
8. Architecture Assets
9. Known Technical Debt
10. Risks
11. Lessons Learned
12. Next Roadmap
13. Release Approval

## Naming

`docs/releases/vX.Y.Z-[status].md`
